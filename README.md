# DoeFacil
Plataforma de doações transparentes para ONGs utilizando Ethereum e Solidity.

## Objetivo:

Permitir que ONGs verificadas recebam doações em ETH com total transparência através da blockchain.

 ## Tecnologias:
 
- Solidity
- Hardhat
- React
- Ethers.js
- MetaMask
- Sepolia Testnet
